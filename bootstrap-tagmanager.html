Tagmanager Example Usage

Basic Usage

  • HTML
  • JS
<input type="text" name="tags" class="tagManager"/>

<script>
    $(".tagManager").tagsManager();
</script>

Custom tagInput

  • HTML
  • JS

Warning!

id attribute is highly required 
<input type="text" id="tags" class="tagManager2"/>

<script>
    $(".tagManager2").tagsManager({
        inputTag:'<input type="text" class="input-medium search-query">'
    });
</script>

Custom tagsParent element

  • HTML
  • JS
<div class="well" style="background:#FFE491">
    <div id="myCustomTagsParent"></div>
</div>
<p class="well">            
    <input class="tagManager3" name="tags" type="text"/>
</p>

<script>
    $(".tagManager3").tagsManager({
        tagsParent:'#myCustomTagsParent'
    });
</script>

Custom tagInsert method(before,after,append,prepend)

  • HTML
  • JS
<div id="myCustomTagsParentAppend" class="well" style="background:#FEE491"></div>
<p class="well">            
    <input class="tagManager4" name="tags" type="text"/>
</p>

<script>
    $(".tagManager4").tagsManager({
        tagInsert:'prepend',
        tagsParent:'#myCustomTagsParentAppend',
        backspace:[46] // use DELETE key instead BACKSPACE
    });
</script>
